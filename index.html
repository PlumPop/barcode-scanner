Initial Setup

  1. Reset the barcode scanner to the default factory settings.
  2. Enable the "Change Bluetooth Name" mode.
  3. Set the barcode scanner name to "PlumPop Scanner".
  4. Save the "PlumPop Scanner" name.
  5. Update the standby setting to never.
  6. Test the barcode scanner works with a PlumPop Merchant tablet.
  7. Turn off the volume.
  8. Standby immediately.
